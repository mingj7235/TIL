# ⚓️ 왜 쿠버네티스인가?

컨테이너를 쉽고 빠르게 배포/ 확장하고 관리를 자동해주는 오픈 소스


Google에서 사용하던 borg 를 기반으로 만든 오픈소스

CNCF (Cloud Native Computing Foundation) 소속 즉, 완전한 오픈소스


Planet Scale (구글 스케일보다 적게 사용하면 ok..)


무한한 확장성
    
    - 머신러닝, CI/CD, 서비스메시, 서버리스 등등.. 
    
    - 쿠버네티스 내에서 엄청난 확장성을 가지고 서비스들이 동작된다.


사실상의 표준 (de facto) 

    - 컨테이너 오케스트레이션의 사실상 표준이 쿠버네티스다. 

    - 심지어 Docker 마저 쿠버네티스를 지원

    - Amazon - EKS 

    - 사실상의 표준이라는 것은, 쓰임이 많으며 엄청나게 많은 라이브러리가 제공된다는것.


CNCF 란?

- 클라우드 환경에 적합한 컴퓨팅 기술을 지원해주는 오픈소스 단체.

- Conainerization, CI / CD, Orchestration, Monitoring, Service discovery 등등

- 쿠버네티스가 Cloud Native의 핵심역할을 한다.
