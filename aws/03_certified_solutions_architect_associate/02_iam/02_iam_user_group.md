# IAM - User, Group

## 암호 정책 설정

IAM -> 계정 설정 -> 편집

    - 암호 정책에 대한 설정을 할 수 있다.

    - 암호 길이, 암호 패턴, 암호만료 활성화등을 설정할 수 있다.

## User, Group 생성

IAM 을 통해 User 를 생성하고, 그룹을 생성하여 그룹에 권한을 주고, User 를 관리할 수 있다.

Login 시, IAM 으로 로그인할 때, 계정 별칭을 사용하여 로그인 해야한다. 

기본적으로 그룹의 권한이 부여되지만, User 개인에게도 권한을 부여할 수 있다.
